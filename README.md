# QuizOnline

#Introduction
This is an application that helps teachers to take tests, compose questions into Bank and manage students in their class.

# Technology
 1. ASP.NET MVC 4.7
 2. SQL Server 2019
 3. Boostrap
 4. Jquery
 5. Ajax
# How to run
 1. Run Script Db in Resource
 2. Change connection String in Source Code
 3. Run Project
